# GitHub Pages Setup for WispMark

Repo: `https://github.com/abarenbe/WispMark`

## 1) Push these files

Commit and push:
- `docs/index.html`
- `docs/support.html`
- `docs/privacy.html`

## 2) Enable GitHub Pages

1. Open repository settings:
   - `https://github.com/abarenbe/WispMark/settings/pages`
2. Under **Build and deployment**:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/docs`
3. Save.

GitHub will publish to:
- `https://abarenbe.github.io/WispMark/`

## 3) URLs for App Store Connect

Support URL:
- `https://abarenbe.github.io/WispMark/support.html`

Privacy Policy URL:
- `https://abarenbe.github.io/WispMark/privacy.html`

## 4) Optional custom domain later

If you buy a domain, keep this pages content and point the custom domain to GitHub Pages.
