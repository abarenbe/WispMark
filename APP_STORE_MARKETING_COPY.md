## Description
WispMark is a fast, floating Markdown notes app for macOS.

Capture thoughts instantly from your menu bar, keep your notes local on your Mac, and organize everything with simple plain-text workflows.

Features:
- Floating, always-ready note window
- Local-only storage (no account required)
- Markdown editing with clean live formatting
- Wiki links with `[[Note Name]]` navigation
- Tagging with `#tags` and quick filtering
- Code blocks with copy support
- Pin important notes and restore recently deleted notes
- Keyboard-friendly workflow with global hotkeys


Built for people who want quick capture and durable plain-text notes without cloud complexity.

- No account, sync service, or external API is required.

- Global hotkeys are used for quick app toggle/new note actions.
