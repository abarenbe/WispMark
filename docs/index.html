WispMark for macOS

Capture ideas instantly.
Keep notes local.

WispMark is a fast, floating markdown notes app for Mac. Write in plain text, link notes together, organize with tags, and keep full control of your files.

Mac App Store listing coming soon Support Privacy Policy

Local-first by design

No account required. Notes stay on your Mac and are saved as durable markdown files.

Built for speed

Open quickly from the menu bar and capture thoughts without breaking your flow.

Markdown + wiki links

Use headings, lists, checkboxes, code blocks, and [[wiki links]] to connect ideas.

What you can do in WispMark

  • Create and search notes from one floating window.
  • Use #tags and workspaces to organize quickly.
  • Pin critical notes and restore recently deleted notes.
  • Write and copy code blocks in clean markdown format.
  • Import notes from an existing local notes folder.
