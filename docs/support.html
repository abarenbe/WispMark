WispMark Support

If you need help with WispMark, use the options below.

Contact

Open a support issue on GitHub:
https://github.com/abarenbe/WispMark/issues/new/choose

Include this information

  • macOS version
  • WispMark version and build (shown in app menu)
  • Steps to reproduce
  • Screenshot if relevant

Data Location

WispMark stores notes locally on your Mac. In current local builds, notes are stored in:
~/Library/Application Support/WispMark/Notes/
